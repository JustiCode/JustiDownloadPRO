# JustiDownload PRO 1.1

**JustiDownload PRO** es un software portable diseñado por **Mauro Ocampo (JustiCode)** bajo la filosofía de "Accesibilidad Universal". El objetivo es romper las barreras en la descarga de contenido multimedia para personas con discapacidad visual.

## 🛠 Características Principales
* **100% Accesible:** Optimizado para NVDA y JAWS.
* **Formatos:** Descarga en MP3, MP4, M4A y WAV.
* **Sin Anuncios:** Interfaz limpia, nativa y sin distracciones.
* **Portable:** No requiere instalación compleja.

## 🚀 Instalación y Uso
1. Descargá el programa compilado con el ejecutable dentro desde la sección de **Releases**.
  2. Asegurate de tener `ffmpeg.exe` en la misma carpeta para las conversiones de audio.
3. Ejecutá con privilegios de administrador y empezá a descargar en el directorio por defecto o personaliza uno a elección.
