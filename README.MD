# JustiDownload PRO 1.0 🚀
### El Descargador Multisitio Definitivo con Accesibilidad Universal

**JustiDownload PRO** es una herramienta de escritorio portable desarrollada por **Mauro Ocampo (JustiCode)**. Este software permite la descarga de contenido multimedia de cientos de sitios web (YouTube, Facebook, Instagram, TikTok, etc.) priorizando la accesibilidad total para usuarios de lectores de pantalla como **NVDA y JAWS**.

---

## ✨ Características Principales
- **Multisitio:** Gracias al motor `yt-dlp`, soporta una vasta gama de plataformas sociales y de noticias.
- **Accesibilidad Nativa:** Interfaz construida en `wxPython`, garantizando que todos los elementos sean detectados correctamente por tecnologías asistivas.
- **Control por Teclado:** Operación completa mediante atajos (Alt + Letra) para una navegación sin ratón.
- **Formatos Profesionales:** Descarga y conversión directa a MP3, MP4, M4A y WAV utilizando `FFmpeg`.
- **Portable:** No requiere instalación. Ejecución directa desde el archivo .exe.

---

## ⌨️ Atajos de Teclado Rápidos
| Tecla | Acción |
|-------|--------|
| **Alt + B** | Iniciar búsqueda |
| **Alt + D** | Descargar seleccionado |
| **Alt + R** | Reproducir en navegador/reproductor |
| **Alt + I** | Información del video |
| **Alt + 1 / 2** | Cambiar entre pestañas |
| **F5** | Detener reproducción inmediata |

---

## 🛠️ Requisitos del Sistema
- **S.O.:** Windows 10 o superior (recomendado).
- **Conexión:** Acceso a internet activo.
- **Permisos:** Se recomienda ejecutar en una carpeta con permisos de escritura (como Documentos o una carpeta dedicada en C:\).
